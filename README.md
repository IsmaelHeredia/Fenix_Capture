Fenix Capture
=============

A screenshot made in C# that has the following options:<br/>

[+] Capture full screen<br/>
[+] Capture windows<br/>
[+] Set time and name of the capture<br/>
[+] Capture forms of any process (DEPRECATED)<br/>
[+] Store all images in logs<br/>
